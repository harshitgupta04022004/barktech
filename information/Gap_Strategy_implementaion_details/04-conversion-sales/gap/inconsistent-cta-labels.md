# Gap: Inconsistent CTA Labels and Behavior

## Current state

Product CTAs use mixed, confusing labels and behaviors:

| Label seen | Behavior |
|------------|----------|
| Send Inquiry | Dead (`#`) |
| Request To Call Back | Form (non-functional backend) |
| Get a Price/Quote | Text only on some cards |
| Call for /Price | Inconsistent wording |
| Call: +91… | Works (tel: link) |

No single conversion pattern across 15+ products.

## Impact

- Cognitive friction for buyers
- Analytics cannot compare CTA performance

## Severity

**High**
