# Strategy: No Admin Dashboard

## Objective

Internal dashboard for sales and marketing KPIs.

## MVP metrics

- RFQs this week / by product / by source
- Top search queries
- AI chat: sessions, handoffs to human, top questions
- Form conversion rate by page

## Build options

| Option | When |
|--------|------|
| Zoho CRM reports | Fastest |
| Metabase + PostgreSQL | Custom, full control |
| Retool internal app | Quick admin tools |

## Access

- Role: Sales (leads only), Marketing (analytics), Admin (all)

## Effort

**Medium** (Metabase: 3–5 days)
