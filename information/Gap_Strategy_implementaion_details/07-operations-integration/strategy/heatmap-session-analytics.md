# Strategy: No Heatmap or Session Analytics

## Objective

See click patterns and session recordings on key pages.

## Tool options

| Tool | Cost |
|------|------|
| Microsoft Clarity | Free |
| Hotjar | Freemium |
| FullStory | Paid |

## Pages to instrument first

- Homepage product grid
- Top 3 product detail pages
- Contact / RFQ form

## GA4 goals (complement)

- `click_call`, `click_whatsapp`, `submit_rfq`, `click_send_inquiry`
- Funnel: product view → quote start → quote submit

## Privacy

- Disclose in cookie policy
- Mask form fields in recordings

## Effort

**Low** (1 hour Clarity install)
