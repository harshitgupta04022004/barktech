# Gap: Inconsistent Product Naming

## Current state

Same machines referenced with different names, URLs, and casing across pages.

## Examples

- "semi automatic flute laminator Machine" (inconsistent casing)
- Product images in `/images/products/` vs `/images/product/`
- Some broken internal links from wget (wrong case in filenames)

## Impact

- User confusion
- SEO cannibalization
- Broken assets on mirror/offline deploy

## Severity

**Medium**
