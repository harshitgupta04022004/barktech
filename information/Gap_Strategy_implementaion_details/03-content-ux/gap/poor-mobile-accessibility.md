# Gap: Poor Mobile and Accessibility UX

## Current state

- Legacy responsive CSS (`responsive.css`) but layout issues observed (header wrap, overlapping widgets)
- Search hidden on some breakpoints (`hidden-xs` on desktop header)
- Alt text often empty on product images (`alt=""`)
- Color contrast and keyboard nav not verified

## Impact

- Many Indian buyers browse on mobile
- Accessibility failures exclude users and may affect SEO indirectly

## Severity

**Medium**
