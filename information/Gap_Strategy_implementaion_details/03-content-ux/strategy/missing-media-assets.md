# Strategy: Missing Professional Media Assets

## Objective

Minimum media kit per product; hero video on homepage.

## Per-product minimum

| Asset | Spec |
|-------|------|
| Hero image | 1200×800 WebP, white/neutral background |
| Gallery | 3–5 images (detail, control panel, output samples) |
| Video | 60–90 sec machine running (YouTube embed or self-hosted) |
| PDF thumbnail | Datasheet cover preview |

## Site-level

- Factory tour video (2–3 min)
- Replace missing backgrounds with licensed industrial imagery or real plant photos
- Compress all images; use `next/image` with WebP

## Production shoot

- One-day factory shoot for top 10 products
- Budget: local industrial photographer + video

## Effort

**Medium** (content production, not only dev)
