# Strategy: No Case Studies or Social Proof

## Objective

3–5 published case studies in first quarter; grow to 15+.

## Case study template

1. Client (with permission) + city
2. Problem before Bark machine
3. Machine installed + model
4. Results (speed, waste reduction, ROI if available)
5. Photos of install
6. Quote from plant manager

## Quick wins

- UDYAM registration badge on footer
- "Installed in: Ghaziabad, Noida, Ahmedabad, Pune…" map
- Google reviews + LinkedIn recommendations linked from site

## CMS

- Content type: `CaseStudy` linked to `Product` many-to-many

## Effort

**Medium** (sales + marketing outreach)
