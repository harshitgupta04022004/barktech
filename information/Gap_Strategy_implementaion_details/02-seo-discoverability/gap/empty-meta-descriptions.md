# Gap: Empty Meta Descriptions

## Current state

Most pages have blank or generic `<meta name="description" content="">`.

## Evidence

- `index.html`: empty description
- 15+ product pages: empty description
- Only `about-us.html`, `contact-us.html`, and 1–2 products have text

## Impact

- Poor Google snippet in search results
- Lower CTR from organic search
- Weak ranking for "die cutting machine India", "flute laminator manufacturer"

## Severity

**High** for SEO
