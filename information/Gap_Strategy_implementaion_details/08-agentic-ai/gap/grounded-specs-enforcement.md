# Gap: AI Must Not Invent Specifications

## Current state

No AI assistant exists. Risk: if built naively, LLM would hallucinate speeds, prices, and models — same class of error as hand-edited HTML (FMZ mismatch, Chinese spec paste).

## Principle (from audit)

> Every number the assistant states must come from product database lookup, not LLM training knowledge.

## Severity

**Critical** for AI launch
